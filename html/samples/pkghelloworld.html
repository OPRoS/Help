helloworld package

  • HelloMaker component reads a property message from component profile, and sent this message through three ports.
    • Set a period counter
    • Call print method through the service port MessagePrint at every 3 period
    • Send the message through the data port in every 3 period+1
    • Send the message through the event port at every 3 period+2
  • MessagePeinter component sets monitoring variables and response to HelloMaker component
    • Increment the monitoring variable at every period
    • Provide print method
    • Read message through the data port
    • Response to the event
