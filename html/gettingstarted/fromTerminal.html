Execute the helloworld Application on the Command Window

  1. Double-click the run.bat at OPRoS\OprosRobot\Binary
  2. Type "pkg run" at the cursor on the command window. Then press the Enter key.
  3. The helloworld, a default application, is executed.
  4. Type "pkg stop" at the cursor in the command window. Next, press the Enter key.
  5. The application being executed is terminated.
