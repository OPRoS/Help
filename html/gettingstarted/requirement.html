Requirements for Installation

  • Operating System: Windows7 32bit
  • Required Disk Space: 1.78GB
  • Required Programs:
    •   Visual Studio 2010 (possible to use a demo version), or  VC++2010 Redistributable Package
    •   VC++2010 SP1 (for OPRoS Tools and Simulator)
    •   JAVA 6.0 or above (for OPRoS Tools)
    •   MSXML4msms.exe (for OPRoS Simulator)
